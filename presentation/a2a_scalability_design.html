✨ 확장성의 장점

  • ✓
    기존 코드 수정 불필요
    새 에이전트 추가 시 다른 코드 변경 없음
  • ✓
    독립적 배포 가능
    각 에이전트를 개별적으로 배포/업데이트
  • ✓
    장애 격리
    한 에이전트 문제가 전체 시스템에 영향 없음
  • ✓
    자동 서비스 발견
    레지스트리가 새 에이전트를 자동으로 관리
  • ✓
    플러그인 아키텍처
    새 기능이 플러그인처럼 쉽게 추가됨

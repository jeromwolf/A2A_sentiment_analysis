A2A (Agent-to-Agent Protocol)

구글이 2025년 4월에 제시한 서로 다른 기술로 구축된 에이전트 간 통신 프로토콜입니다.

🎯목적

A2A 프로토콜은 이를 그대로 수많은 AI 에이전트들 간에 서로 소통 및 상호작용하고 협력할 수 있게 만들어주는 표준화된 규칙을 말합니다.

⚡특징

P2P 통신 중앙 집중식 제어 없음
확장성 유연한 구조
장애 격리 안정적 운영

핵심 개념

"에이전트들이 서로 대화하며 문제를 해결"

단순한 메시지를 주고받는 것을 넘어 다른 에이전트의 존재를 인지하고,
능력을 파악하며 작업을 요청하고 결과를 교환하는 전 과정을 표준화

🔗 이기종 프레임워크 통합

에이전트들이 만들 수 있는 다양한 프레임워크:
LangGraph, AutoGen, CrewAI, Python 기반 프레임워크

💡A2A의 가치

• 표준화된 통신: 어떤 프레임워크로 만들어도 소통 가능

• 자율적 협업: 중앙 제어 없이 에이전트끼리 직접 협업

• 동적 발견: 새로운 에이전트를 실시간으로 찾고 활용

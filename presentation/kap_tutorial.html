🎬 Kap으로 A2A 애니메이션 GIF 만들기

1

Kap 실행하기

  • Applications 폴더에서 Kap 아이콘 클릭
  • 또는 Spotlight 검색 (⌘ + Space)에서 "Kap" 입력
💡 첫 실행 시: 화면 녹화 권한을 요청할 수 있습니다. 시스템 환경설정 → 보안 및 개인 정보 보호 → 화면 녹화에서 Kap 허용
2

브라우저에서 애니메이션 열기

  • Chrome이나 Safari에서 a2a_smithery_style.html 파일 열기
  • 전체 화면으로 보기 권장 (더 깨끗한 GIF 생성)
3

녹화 영역 선택하기

🖱️ Kap이 실행되면 화면에 점선 프레임이 나타납니다

  • 프레임 모서리나 변을 드래그해서 크기 조절
  • 프레임 안쪽을 드래그해서 위치 이동
  • 애니메이션 영역만 정확히 선택 (하단 정보 박스 포함)
💡 팁: 정확한 영역 선택을 위해 프레임 크기를 조금 크게 잡은 후, 모서리를 드래그해서 미세 조정하세요.
4

녹화 시작하기

  • 화면 중앙 하단의 ● 녹화 빨간 버튼 클릭
  • 또는 단축키: ⌘ + Shift + 5
  • 3초 카운트다운 후 녹화 시작
⚠️ 중요: 애니메이션이 한 사이클 완료될 때까지 기다리세요 (약 20-30초)
5

녹화 중지하기

  • 메뉴 바 상단의 ■ 정지 버튼 클릭
  • 또는 Touch Bar가 있다면 정지 버튼 터치

녹화가 끝나면 자동으로 편집 창이 열립니다.

6

GIF로 내보내기

편집 창이 열리면:

  1. 우측 상단의 포맷 선택에서 GIF 선택
  2. GIF 설정 조정:
    • FPS: 15-20 (기본값 사용 권장)
    • 크기: 원본 또는 축소 (파일 크기 고려)
    • 품질: 높음 (파일이 너무 크면 중간)
  3. 하단의 ✓ 저장 버튼 클릭
  4. 파일명 입력 (예: a2a_animation.gif)
  5. 저장 위치 선택 후 저장

🎯 최종 체크리스트

  • ✅ GIF 파일 크기가 10MB 이하인지 확인
  • ✅ 애니메이션이 부드럽게 반복되는지 확인
  • ✅ 텍스트가 읽기 쉬운지 확인
  • ✅ 필요시 크기 조정하여 다시 내보내기
🚀 프레젠테이션 활용:
  • PowerPoint/Keynote에 GIF를 삽입하면 자동 재생됩니다
  • 슬라이드 쇼 모드에서도 계속 반복됩니다
  • 파일 크기가 크면 별도 슬라이드로 분리 권장
문제 해결:
  • Kap이 보이지 않는다면: 메뉴 바 상단을 확인하세요
  • 녹화가 안 된다면: 시스템 환경설정에서 화면 녹화 권한 확인
  • GIF가 너무 크다면: 크기를 50%로 줄이거나 FPS를 낮추세요

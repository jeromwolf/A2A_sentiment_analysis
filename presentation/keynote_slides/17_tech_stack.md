# 슬라이드 17: 기술 스택

## 사용된 기술들

### Backend
- **Python 3.11**: 메인 언어
- **FastAPI**: 비동기 웹 프레임워크
- **WebSocket**: 실시간 통신
- **asyncio**: 비동기 처리

### AI/ML
- **Gemini AI**: 감성 분석
- **yfinance**: 주가 데이터
- **ta**: 기술적 지표

### 인프라
- **Docker**: 컨테이너화
- **Redis**: 캐싱 (선택사항)
- **uvicorn**: ASGI 서버

### 외부 API
- NewsAPI, Finnhub, Twitter API v2, SEC EDGAR

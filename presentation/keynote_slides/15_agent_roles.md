# 슬라이드 15: 에이전트 구성

## 11개 전문 에이전트의 역할

### 데이터 수집 (4개)
- **NLU Agent**: 자연어 → 티커 추출
- **News Agent**: 뉴스 데이터 수집
- **Twitter Agent**: 소셜 미디어 감성
- **SEC Agent**: 기업 공시 자료

### 분석 처리 (5개)
- **Sentiment Agent**: 감성 분석 (Gemini AI)
- **Quantitative Agent**: 기술적 지표 분석
- **Score Agent**: 가중치 기반 점수 계산
- **Risk Agent**: 리스크 평가
- **Report Agent**: 최종 리포트 생성

### 인프라 (2개)
- **Registry Server**: 에이전트 관리
- **MCP Agent**: 외부 도구 연동
